### Introdução
> Desenvolver um lustre com 3 LEDs, um sempre ligado, outro piscando a cada certo intervalo de tempo e outro ativado por um botão.
### Lista de materiais
* 1 Protoboard
* 3 LEDs
* Um interruptor
* Cabos jumper macho-fêmea, macho-macho.
### Plataforma escolhida:
> *Tinkercad*
